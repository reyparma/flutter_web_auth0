# flutter_web_auth0

Flutter Web Authentication Using Auth0, BloC and Go_Router

There are a lot of Flutter authentication examples using BloC and Go_Router with Firebase or Google IAM.  This Flutter web application works with Auth0 for authentication, BLoC for state management and Go_Router for navigation.
